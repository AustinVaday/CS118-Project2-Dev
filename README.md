# CS118-Project2-Dev
The purpose of this project is to use UDP Socket and C/C++ programming language to implement a reliable data transfer protocol similar to that in TCP.

Authors: Yingbo (Max) Wang and Austin Vaday

We've designed a TCP header that looks as follows:
Source: http://www.freesoft.org/CIE/Course/Section4/8.htm
Our implementation as a struct (in tcp.h) does not include the data field.
We append that data after serializing the struct.
